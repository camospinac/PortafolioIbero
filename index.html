Proyectos recientes

Nos apasiona lo que hacemos; conoce nuestros proyectos más recientes.

Sistema de Inventarios

Una aplicación de escritorio diseñada para pequeñas empresas que permite el registro, seguimiento y alerta de stock mínimo de productos en tiempo real.

Conoce más

“El mayor reto fue diseñar una arquitectura que permitiera manejar múltiples registros sin que el sistema perdiera rendimiento. Lograr la persistencia de datos fue una prueba de fuego para nuestras habilidades de lógica.”

Nicolás Barajas Desarrollador de Software

Plataforma de Tutorías

Web app diseñada para conectar estudiantes de la Ibero de semestres superiores con nuevos alumnos para tutorías personalizadas en materias de ciencias básicas.

Conoce más

“Aprendí que la tecnología es solo una herramienta; lo más valioso fue entender cómo el diseño centrado en el usuario puede facilitar el aprendizaje colaborativo dentro de nuestra comunidad universitaria.”

Nicolás Gómez Desarrollador Web

Calculadora de Salud Financiera

Herramienta digital para ayudar a jóvenes profesionales a proyectar sus ahorros, gestionar deudas y entender el impacto de las tasas de interés en sus finanzas personales.

Conoce más

“Al principio, el manejo de excepciones y errores de entrada de usuario nos daba muchos problemas. Solventamos esto implementando validaciones rigurosas y un sistema de feedback claro para que la experiencia sea fluida.”

Camilo Ospina Desarrollador Software
